## Заливання на GitHub

Заходжу в папку Тask і створюю папку task-0, добаляю в неї файли з першого завдання, Створюю папку task-1 і добавляю в неї файл README.md, роблю коміт і пушу, але перед цим довелочсь прописати `git pull`ця команда працює як комбінація команд `git fetch` і `git merge`. Git спочатку забирає зміни із зазначеного віддаленого сховища, а потім намагається злити їх з поточної гілкою. Ну а вже потім успішний пуш
